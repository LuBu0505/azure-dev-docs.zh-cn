#### <a name="inventory-passwords-and-secure-strings"></a>清单密码和安全字符串

检查生产部署上的所有属性和配置文件以及所有环境变量中是否有机密字符串和密码。 在 Spring Boot 应用程序中，通常可以在 application.properties 或 application.yml 文件中找到此类字符串 。
