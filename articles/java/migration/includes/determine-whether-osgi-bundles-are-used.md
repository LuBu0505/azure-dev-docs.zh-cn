### <a name="determine-whether-osgi-bundles-are-used"></a>确定是否使用了 OSGi 捆绑

如果使用了添加到 WebLogic 服务器的 OSGi 捆绑，则需将等效的 JAR 文件直接添加到 Web 应用程序。
