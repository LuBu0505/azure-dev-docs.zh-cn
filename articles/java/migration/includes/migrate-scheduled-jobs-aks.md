### <a name="migrate-scheduled-jobs"></a>迁移计划的作业

若要在 AKS 群集上执行计划的作业，请按需定义 Kubernetes CronJob。 有关详细信息，请参阅[使用 CronJob 运行自动化任务](https://kubernetes.io/docs/tasks/job/automated-tasks-with-cron-jobs/)。
