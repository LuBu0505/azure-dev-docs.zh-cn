### <a name="inventory-server-capacity"></a>清点服务器容量

记录当前生产服务器的硬件（内存、CPU、磁盘），以及平均和峰值请求计数和资源利用率。 此信息必须告知所选 VM 大小。 有关详细信息，请参阅[云服务的大小](/azure/cloud-services/cloud-services-sizes-specs)。
