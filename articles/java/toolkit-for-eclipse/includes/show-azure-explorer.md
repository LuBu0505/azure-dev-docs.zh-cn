## <a name="displaying-the-azure-explorer-view"></a>显示 Azure 资源管理器视图

如果“Azure 资源管理器”视图**** 未显示在 Eclipse 中，请通过以下步骤来显示它：

1. 单击“窗口”，将鼠标悬停在“显示视图”上，然后单击“其他”  。

2. 在“显示视图”对话框中，展开“Azure”，选择“Azure 资源管理器”，然后单击“打开”   。
