完成本教程后，需要删除创建的两个资源，以确保不会向你收费。 

1. 在 Visual Studio Code 中，使用用于数据库的 Azure 资源管理器，右键单击资源并选择“删除帐户…”。
1. 在 Visual Studio Code 中，使用用于应用服务的 Azure 资源管理器，右键单击资源并选择“删除”。
