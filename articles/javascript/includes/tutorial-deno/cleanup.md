在本部分中，我们将删除和清理所有已创建的资源。

创建的应用服务包括一个在免费定价层上运行的后备应用服务计划，因此你不会产生任何持续成本。

需要清理资源时，请访问 [Azure 门户](https://portal.azure.com)，选择“资源组”，找到并选择在本教程中创建的资源组（例如 `deno-quickstart`），然后使用“删除资源组”命令。
