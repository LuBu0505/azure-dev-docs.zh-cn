# <a name="part-6-clean-up-resources"></a>第 6 部分：清理资源

[上一步：进行更改并重新部署](tutorial-vscode-static-website-node-05.md)

你创建的存储会产生很少的后续费用（对于存储而言，非常少）。 若要避免产生这些费用，请在Visual Studio Code 的“Azure:  存储”资源管理器中右键单击存储帐户，然后选择“删除存储帐户”  。
